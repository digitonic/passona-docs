# Senders

Senders are what the end recipients handset sees as who send the message. {{ className: 'lead' }}

All teams will have a default sender of `88440` as well as a stripped down version of their team name up to a maximum of 11 characters. {{ className: 'lead' }}

Any new senders created must be between **3-11** characters in length. {{ className: 'lead' }}

---

## Create Sender {{ tag: 'POST', label: '/api/2.0/senders/' }}

<Row>
  <Col>
      Create a new sender by providing a **3-11** character length string.
  </Col>
  <Col sticky>

    <CodeGroup title="Request" tag="POST" label="/api/2.0/senders/">

    ```bash {{ title: 'cURL' }}
    curl --location 'https://passona.co.uk/api/2.0/senders/' \
    --header 'Authorization: Bearer B1gooyw1FlLcITdn2nd4vfem4b4YYogJUGjVTnrrSTOt2T5hCFnS2xpANsYd' \
    --header 'Accept: application/json' \
    --header 'Content-Type: application/json' \
        --data '{
        "sender": "NewSender"
    }'
    ```
    </CodeGroup>

    ```json {{ title: 'Response' }}
      {
          "data": {
              "uuid": "ae1fc0f8-fb28-11e9-87dc-0242ac120008",
    ```
